Внешняя обработка для индивидуализации КИЗ
==========================================

Внешняя обработка 1С:Предприятие 8.3 для RFID индивидуализации контрольных
идентификационных знаков используемых в ГИС Маркировка.


Чтобы сбросить заблокированные метки у эмулятора RFID устройства, необходимо
отправить ему специальную команду ReinitializeTransponders через
[Менеджер Шины RFID](http://rfidcenter.ru/files/RfidBusManagerSetup.exe), или
[через 1С](https://github.com/RfidCenter/RfidBus/tree/master/1C/Readers.SpecialCommands).

Статья: http://rfidcenter.ru/blog/gis_markirovka_individualizacija_kiz_iz_1c

Другие статьи о работе с Шиной RFID из 1С: http://rfidcenter.ru/blog/tag/1c
